This example is based on the GD32VF103V-EVAL-V1.0 board, it provides a description of how to use Tamper key to control the LED1. The example uses polling mode.

Press the KEY_B, LED1 will be on. Press the KEY_B again, LED1 will be off.

On the GD32VF103V-EVAL-V1.0 board, the Tamper key is connected to PA0 and the LED1 is connected to PC0.
