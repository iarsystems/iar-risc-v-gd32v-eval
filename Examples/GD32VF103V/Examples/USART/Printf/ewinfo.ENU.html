Information Center for RISC-V | EXAMPLES

This demo is based on the GD32VF103V-EVAL-V1.0 board, it shows how to retarget the C library printf function to the USART.

The USARTx is configured as follow:
- BaudRate = 115200 baud
- Word Length = 8 bits
- One Stop Bit
- No parity
- Hardware flow control disabled (RTS and CTS signals)
- Receive and transmit enabled

JP5 and JP6 must be fitted.
